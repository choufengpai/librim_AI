## 七、Agent 协作说明

### 如何使用此文档

1. **开始任务**: 将任务状态从 ⏳ 改为 🚧，在「执行者」列填写自己的标识
2. **完成任务**: 将状态改为 ✅
3. **遇到阻塞**: 将状态改为 ❌，并在「四、阻塞项」中说明
4. **添加新任务**: 在相应章节添加新行
5. **记录决策**: 在「五、决策记录」中添加重要决策

### 任务格式

```
| ID | 任务描述 | 状态 | 说明 | 执行者 |
|----|----------|------|------|--------|
| T0XX | 任务内容 | ⏳ | 补充说明 | agent名称 |
```
